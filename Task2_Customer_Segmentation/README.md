# Customer Segmentation Analysis

## Objective
Segment customers based on purchasing behaviour using RFM Analysis and K-Means clustering.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
Online Retail Dataset (UCI/Kaggle)

## Features
- Data Cleaning
- RFM Analysis
- StandardScaler
- Elbow Method
- K-Means Clustering
- Cluster Visualization
- Customer Profiling
- Marketing Recommendations

## Results
The model identified four customer segments:
- Regular Customers
- At-Risk Customers
- VIP Customers
- Loyal Customers
